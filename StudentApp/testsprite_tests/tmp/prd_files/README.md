# StudentApp Frontend PRD Source

This project is a React Native Expo StudentApp that supports both mobile and web targets.

Core user flows:

- User login and logout.
- View students list.
- Search and sort students.
- Add a student.
- Edit a student.
- Delete a student with confirmation.
- Refresh list and recover from network errors.

UX and quality expectations:

- Responsive behavior for mobile and web layouts.
- Accessible controls with labels/hints.
- Modal form validation and unsaved-change protection.
- Robust handling of API errors and retries.
